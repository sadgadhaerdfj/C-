# 哈弗曼编码

### 哈夫曼树

给定n个权值作为n个叶子结点，构造一棵二叉树，若该树的带权路径长度达到最小，则称该二叉树为哈夫曼树，也被称为最优二叉树。

构建思路
下面给出一个非常简洁易操作的算法，来构造一棵哈夫曼树：
 1、初始状态下共有n个结点，结点的权值分别是给定的n个数，将他们视作n棵只有根结点的树。
 2、合并其中根结点权值最小的两棵树，生成这两棵树的父结点，权值为这两个根结点的权值之和，这样树的数量就减少了一个。
 3、重复操作2，直到只剩下一棵树为止，这棵树就是哈夫曼树。

### 构建树的节点

思路：先将权重的键—值对分开，键值对应的存入两个有序的数组，然后把前n个节点进行赋值，也就是树的每个节点的权重和名字，后面的节点赋值为空，这些是组合成的节点，同时哈弗曼编码还是最优前缀编码，对于包括n个字符的数据文件，分别以它们的出现次数为权值构造哈夫曼树，则利用该树对应的哈夫曼编码对文件进行编码，能使该文件压缩后对应的二进制文件的长度最短

```c
void Huffman(HuffmanTree p)//用来创造树的结构
{
    int leaf[n];int s=0;char leaf_name[n][4];    //leaf叶,需要处理的的数字,leaf_name为字符
    char * temp=strtok(str,",");

    {   //printf("%d :%s\n",i,temp);
        if(i%2==1){leaf[s++]=atoi(temp);}   //数字放进数组 
        else{strcpy(leaf_name[s],temp);}   //字符放进数组
        temp=strtok(NULL,",");
    }
    //for(int i=0;i<n;i++){printf("%s  %d\n",leaf_name[i],leaf[i]);}      //用来检测叶的数组

    for(int i=0;i<2*n-1;i++)
    {
        p[i].parent=p[i].lchild=p[i].rchild=-1;     //树的初始化
        if(i<n)                 //前n个结点赋值,后面的全部置0
        {
            p[i].weight=leaf[i];            
            strcpy(p[i].name,leaf_name[i]);  
        }
        else{p[i].weight=0;strcpy(p[i].name,"\0");}   //没有名字的给0
    }
}
```

### 建成哈夫曼树

思路：找到哈夫曼树节点里的最小的两个值（compare函数），同时这两个值是没有父节点的，这里我直接设了两个很大的值来比较，如果两个值相等，那么就看子节点来比较，有子节点的大于没子节点的，把找到的两个节点组成一个新的节点，然后重复这个操作 n - 1 次，

```c
void compare(HuffmanTree p)//用来得到权重最小的叶
{
    m1=999998;m2=999999; 

    for(int i=0;i<2*n-1;i++)
    {    
        if(p[i].weight!=0 && p[i].parent==-1)    //挑选出没有父结点的权重最小的叶
        {
            if(p[i].weight<=m1){m2=m1;n2=n1;m1=p[i].weight;n1=i;}
            else if(m1<p[i].weight && p[i].weight<m2){m2=p[i].weight;n2=i;}
        }
    }
    if(m1==m2 && p[n2].lchild==-1 && p[n1].lchild!=-1){int n;n=n1;n1=n2;n2=n;}
} 

void two_min(HuffmanTree p)  
{
    for(int o=0;o<n-1;o++)
    {
    compare(p);
    int i=0;
    while(i<2*n-1)
    {//printf("%d  %d  %d\n",m1,m2,p[i].weight);
        if(p[i].weight==0)          //选择后相加存入
        {
            p[i].weight=m1+m2;
            p[i].lchild=n1;
            p[i].rchild=n2;
            p[n1].parent=i;
            p[n2].parent=i;//printf("%d  %d  %d---\n",n2,p[i].weight,p[n1].parent);
            break;}i++;   //存入后退出该循环
        }
    } 
}
```

### 求哈弗曼编码

思路：从下往上遍历，直到找到根节点，如果是父节点的左孩子，编码就是0，如果是右节点编码就是1，这样记录得到的结果就是该字符的哈夫曼编码值，可以将得到的每个字符的哈弗曼编码存入文本文件中，需要的时候读出来，遍历每个叶子结点就可以得到所有叶子结点的哈弗曼编码值![](C:\Users\磊\Desktop\哈夫曼\2.png)

### 解压

思路：解压过程和压缩过程基本一样，将压缩的二进制文件读出来，每次取一个字节，因为我们是按一个字节长度存进去的，按照压缩的方式反过来还原为哈夫曼编码，将哈弗曼编码转换为对应的字符完成解压

### 哈夫曼编码

哈夫曼树是一棵最优二叉树，而哈弗曼编码的任一个编码都不是其他任何编码的前缀(最左子串），所以每个编码只有一个字符与其对应，所以可以用来压缩和解压
